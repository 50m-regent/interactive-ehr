## プロジェクト概要

インタラクティブEHR（電子カルテ）の研究プロジェクト。現在は初期セットアップ段階。

## 開発環境

- Python 3.12（`.python-version`で管理）
- `uv`でプロジェクト管理（`pyproject.toml`使用、`requirements.txt`なし）
- 実行: `uv run python main.py`
